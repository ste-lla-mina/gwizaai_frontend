# ⚡ Gwiza AI | Intelligent Talent Recruitment

Gwiza AI is a high-performance recruitment dashboard designed to eliminate hiring bias and accelerate candidate screening using ethical AI. Built with **Next.js 14**, **Tailwind CSS**, and **Lucide Icons**.

## ✨ Key Features

* **AI Insights Dashboard:** Real-time analysis of candidate batches.
* **Leaderboard System:** Automated top-10 ranking based on custom skill-matching scores.
* **Skill-Gap Analysis:** Visual breakdowns of candidate strengths vs. job requirements.
* **Responsive Talent Pool:** A mobile-optimized interface for managing candidates on the go.
* **Security First:** Built-in email verification and secure account management flows.

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Styling:** Tailwind CSS (Custom brand palette)
* **Icons:** Lucide React
* **Animations:** Tailwind Animate & Framer Motion transitions
* **Deployment:** Vercel

## 🚀 Getting Started

Install the dependencies:

```bash
npm install
```
Then run 
```bash
npm run dev
```
Open http://localhost:3000 to view the results.
